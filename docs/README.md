# Anonymous Question System Docs

本文档目录用于沉淀匿名提问与大会答疑系统的需求定义、页面文案、数据接口和前端线框说明。

## Documents

- [Product PRD](./product-prd.md)
- [Page Copy](./page-copy.md)
- [Data Model And API](./data-model-and-api.md)
- [Frontend Wireframes](./frontend-wireframes.md)
- [Interaction Design](./interaction-design.md)

## Scope

当前文档默认基于以下前提：

- V1 先做 `H5 网页`，不做微信小程序。
- 处理方式分为 `公开问题`、`会上公开`。
- `原始问题不可编辑、不可改写`。
- 热度依据为 `同类独立提交数`，不做点赞或支持。

## Suggested Reading Order

1. 先读 [Product PRD](./product-prd.md)，统一产品边界和原则。
2. 再读 [Page Copy](./page-copy.md)，确认用户看到的文案和提示。
3. 接着读 [Data Model And API](./data-model-and-api.md)，对齐前后端边界。
4. 再读 [Frontend Wireframes](./frontend-wireframes.md)，理解页面骨架和布局方向。
5. 最后读 [Interaction Design](./interaction-design.md)，推进交互细节、状态设计和开发拆解。
